Student Exam Performance Prediction

This project aims to predict a student's math score based on various input features using machine learning techniques. The input features considered for the prediction include:

  • Gender
  • Race/Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course Completion
  • Writing Score
  • Reading Score

By analyzing these factors, the machine learning model will provide an estimated math score, helping to identify areas where students might need additional support and resources.
